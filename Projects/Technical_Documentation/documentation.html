Introduction

Python is a high-level, general-purpose, and very popular programming language.

Python programming language (latest Python 3) is being used in web development, Machine Learning applications, along with all cutting-edge technology in Software Industry.

Some of notable python features are:

  • Readability: Python's syntax is designed to be clear and readable, making it easy for beginners to learn and understand.
  • Versatility: Python is a versatile language used for various applications, including web development, data science, artificial intelligence, machine learning, automation, and more.
  • Community Support: Python has a large and active community of developers. This community contributes to the wealth of libraries, frameworks, and resources available for Python programming.
  • Interpreted Language: Python is an interpreted language, meaning that code can be executed line by line without the need for compilation. This makes it easier to test and debug code.
  • Object-Oriented: Python supports object-oriented programming, allowing for the creation and use of classes and objects.
Getting Started

Installation:

To start coding in Python, you need to have Python installed on your machine. You can download it from the official Python website.

Hello World Example:

print("Hello, World!")

Running Python Code:

Save the above code in a file with a .py extension (e.g., hello.py) and run it from the command line:

python hello.py
Variables

Variables are containers for storing data values.

Python has no command for declaring a variable.

A variable is created the moment you first assign a value to it.

x = 5 y = "John" print(x) print(y)

Variables do not need to be declared with any particular type, and can even change type after they have been set.

x = 4 # x is of type int x = "Sally" # x is now of type str print(x)
Data Types

In Python, data types are used to categorize and represent different types of values that a variable can hold. Here are some commonly used data types in Python:

  1. Numeric Type:
    • Int: Integer data type represents whole numbers without any decimal points.
    • float: Float data type represents real numbers with decimal points.
    • complex: Complex data type represents complex numbers in the form a + bj, where a and b are real numbers and j is the imaginary unit.
  2. Sring(Str) : String data type represents a sequence of characters, enclosed in single or double quotes.
  3. Boolean Type(bool) : Boolean data type represents the truth values True or False
  4. Sequence Types:
    • list: List is an ordered and mutable collection of items.
    • tuple: Tuple is an ordered and immutable collection of items.
    • range: Range is a sequence of numbers often used for iterating in loops.
  5. Set Types:
    • set: Set is an unordered and mutable collection of unique items.
    • frozenset: Frozenset is an unordered and immutable collection of unique items.
  6. Mapping Type:
    • dict: Dictionary is an unordered and mutable collection of key-value pairs.
  7. NoneType: The None type represents the absence of a value or a null value.
Control Flow Statements

if-else Statements:

if condition: # code to execute if condition is True else: # code to execute if condition is False

elif Statements:

Used to check multiple conditions in a chained manner.

if condition1: # code to execute if condition1 is True elif condition2: # code to execute if condition2 is True else: # code to execute if all conditions are False

while Loop:

while condition: # code to execute while condition is True

for Loop:

for variable in sequence: # code to execute for each element in the sequence

break and continue Statements:

break is used to exit a loop prematurely.

>continue is used to skip the rest of the code inside a loop and move to the next iteration.

for i in range(10): if i == 5: break # exit the loop when i is 5 print(i)

pass Statement:

x = 10 if x > 0: pass # placeholder, no action needed else: print("Non-positive number")
Functions

Python Functions is a block of statements that return the specific task. The idea is to put some commonly or repeatedly done tasks together and make a function so that instead of writing the same code again and again for different inputs, we can do the function calls to reuse code contained in it over and over again.

Defining a Function:

In Python, you define a function using the def keyword, followed by the function name and a set of parentheses. The function may also have parameters (input values) and a return statement (output value).

def greet(name): """This function prints a simple greeting.""" print("Hello, " + name + "!")

Calling a Function:

To use a function, you call it by its name and provide the required arguments.

greet("Alice")

Function with Return Value:

A function can return a value using the return statement.

def add_numbers(x, y): """This function returns the sum of two numbers.""" return x + y

You can capture the returned value when calling the function.

result = add_numbers(3, 5) print(result) # Output: 8

Default Parameters:

You can set default values for function parameters.

def power(base, exponent=2): """This function calculates the power of a number.""" return base ** exponent
